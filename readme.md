# İngilizce Kelime Öğrenme

Bu proje, kullanıcıların İngilizce kelime öğrenmesine yardımcı olmak için tasarlanmış bir web uygulamasıdır. HTML, CSS ve JavaScript kullanılarak oluşturulmuştur ve veritabanı olarak Supabase kullanılmaktadır.

## Özellikler

- Kullanıcı kimlik doğrulama (giriş ve kayıt)
- Kullanıcı ilerleme takibi
- Kelime öğrenme arayüzü
- Quiz ve tekrar etme özellikleri
- Çalışma serisi takibi

## Kullanılan Teknolojiler

- HTML5
- CSS3
- JavaScript (ES6+)
- Firebase (Auth ve Database)

## Kurulum

1. Projeyi bilgisayarınıza klonlayın
2. `index.html` dosyasını bir web tarayıcısında açın
3. Kayıt olun veya giriş yapın
4. Kelime öğrenmeye başlayın!

## Veri Yapısı

Uygulama, Supabase veritabanında aşağıdaki tabloları kullanır:

- `profiles`: Kullanıcı profil bilgileri
- `learned_words`: Kullanıcıların öğrendiği kelimeler
- `user_progress`: Kullanıcı ilerleme bilgileri

## Geliştirme

Bu projede yer alacak gelecek özellikler:

1. İleri seviye kelime öğrenme
2. İstatistik raporları
3. Kelime hatırlatma bildirimleri
4. Arkadaş ekleme ve rekabet özellikleri
