# Proje Kuralları (Project Conventions)

Bu dosya, projenin geliştirme sürecinde uyulması gereken kalıcı kuralları içerir.

## 🚀 GitHub ve Commit Kuralları

1. **Otomatik Push:** Her anlamlı değişiklikten sonra kullanıcı sormadan değişiklikler GitHub'a pushlanmalıdır.
2. **Commit Mesaj Formatı (TR/ENG):** Commit mesajları mutlaka hem Türkçe hem İngilizce olacak şekilde yazılmalıdır.
   - Örnek: `Adam Asmaca ismi Kelime Kurtarma olarak değiştirildi / Renamed Hangman to Word Rescue`
